### What is ALEI?

ALEI replaces the static lead form with a live AI conversation. The moment a prospect submits a form on a business's website, ALEI greets them by name, qualifies their budget, timeline, and pain points — via voice or text chat — and delivers a full buyer profile to the sales team before a human ever picks up the phone.

> *"Stop losing deals before they start."*

### The Problem It Solves

| Pain Point | Reality |
|---|---|
| Slow follow-up | Average lead response time is **42 hours** across industries |
| Wasted calls | Sales reps spend **50% of their week** on unqualified discovery |
| No context | Reps start every call blind with **zero buyer information** |
| Off hours | Leads go cold while your team sleeps |

### Results

| Metric | Impact |
|---|---|
| Discovery time | **75% reduction** |
| Wasted calls | **60% fewer** |
| Time to close | **2x faster** |
| Availability | **24/7 — zero off days** |

### Platform Features

- 🎙️ **Voice & text chat** — Native audio via Google Gemini Live API
- 🧠 **Industry-specific intelligence** — Pre-trained on Real Estate, SaaS, Medical, E-commerce and more
- 🏢 **Multi-tenant dashboard** — Each business gets their own login, lead management, analytics, and CSV export
- 🧪 **Dojo testing environment** — Preview and configure AI behavior before going live
- 🔌 **Embeddable widgets** — Sales AI and Support AI widgets deployable on any website
- 🔗 **CRM integrations** — HubSpot, Salesforce, Zoho (Enterprise tier)
- ⚙️ **Admin panel** — Onboard businesses, manage tenants, assign API keys
